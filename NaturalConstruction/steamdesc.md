[h1]Natural Construction[/h1]

This mod allows you to construct natural tiles and natural backwall with variable mass.

[img]https://raw.githubusercontent.com/Sgt-Imalas/Sgt_Imalas-Oni-Mods/refs/heads/master/Compat_All.png[/img]
[h1]Current latest Version: 1.0[/h1]

[hr][/hr]
[h1]Features[/h1]
[list][*] new Buildings: Natural Backwall and Natural Tile
[*] found in the "Base" build menu category
[*] unlocked by researching "Basic Farming"
[*] default mass is 100kg of any solid material
[*] construction mass can be configured after placing the building blueprint
[*] construction time scales with selected construction mass
[*] default construction mass for both buildings can be configured in the mod config
[*] no AI slop in the mod
[/list]

[h3]Mod is marked as "incompatible" / the game is stuck in an "update loop"?[/h3]
This means that the game failed to install the mod. This is usually caused by Ony Mod Manager breaking the download or OneDrive breaking the games' ability to install mods.
Follow [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3522703974]this guide[/url] to fix the OneDrive issue and replace Ony Mod Manager with Mod Updater if not done already.

you can also grab a direct download on [url=https://github.com/Sgt-Imalas/Sgt_Imalas-Oni-Mods/releases]GitHub[/url] to get the latest version

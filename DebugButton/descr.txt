[h1]Debug Buttons[/h1]

This mod adds a few buttons that give you easier access to the debug mode.
It allows you to activate debug with a simple click instead of typing KLEIPLAY or creating the debug file.

[h1]Current latest Version: 1.0.0[/h1]

[hr][/hr]

[h1]Features[/h1]
[list][*] Debug Mode Button: toggles debug mode on and off. Is highlighted when debug mode is active
[*] Debug Insta Build Button: toggles debug insta build (ctrl+f4) on and off. Is highlighted when insta build is active
[*] Debug Superspeed Button: toggles debug superspeed (x30 speed, Ctrl+U) on and off. Is highlighted when game speed is >x10 
[/list]
